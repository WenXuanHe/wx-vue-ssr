### 搭建vue 服务端渲染

#### 使用vue从零搭建框架，支持以下特性：

```
1.vue + vue-router + vuex  全套解决方案
2.默认使用graphql 请求数据，也可以改为restful
3.利用webpack配置服务端渲染，首屏直出，其他页面异步加载js
4.默认配置typescript管理类型
5.样式使用postcss+cssNext 书写css
6.配置travis持续集成

```

* 使用方式： *
```
npm run server
npm run client
npm run start

```
